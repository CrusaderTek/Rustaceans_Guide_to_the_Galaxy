# Intstall cargo_mobile2
```sh
cargo install --git https://github.com/tauri-apps/cargo-mobile2
```
to update whenever
```sh
  cargo mobile update
```

# Start a project
```sh
mkdir mobile_test
cd mobile_test
cargo mobile init
```
follow the prompts and choose your template project
